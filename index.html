AAHE coursework · group

Metasploitable 2 exploitation

Scanned a deliberately vulnerable Linux target with Nmap, identified an outdated vsFTPd service, and used Metasploit's vsftpd_234_backdoor module to gain a root shell. Confirmed access via whoami/id, then cross-checked the finding's CVSS severity in Nessus. Also ran the Wireshark packet capture and TCP/HTTP header analysis component solo.

Nmap Metasploit Wireshark Nessus
AAHE coursework · group

AWS healthcare security pipeline

Designed and built the security-engineering layer of a cloud migration case study: least-privilege IAM roles, MFA on admin accounts, S3 server-side encryption, and a CloudTrail → CloudWatch → SNS pipeline that alerts on anomalous bucket access. Validated every control end-to-end and documented results against HIPAA/GDPR requirements.

AWS IAM CloudTrail CloudWatch HIPAA / GDPR
AAHE coursework · simulation

Digital forensics: insider threat simulation

A self-generated insider-threat scenario (no real incident or client data) used to practise the forensic workflow: imaged a USB device in FTK Imager, verified MD5/SHA1 hashes, ingested the image in Autopsy for metadata and timeline reconstruction, and cross-validated file timestamps against email and cloud-share logs to confirm a coherent chain of custody.

FTK Imager Autopsy Chain of custody Simulated
